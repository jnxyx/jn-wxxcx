[jn-wxapp](https://github.com/jnxyx/jn-wxxcx) — 一款微信小程序框架（生成器）
==================================================

引言
--------------------------------------

这是一款基于微信小程序开发的程序框架

安装
--------------------------------------

windows用户使用控制台安装:

```bash
$ npm install jn-wxapp -g
```

mac用户使用终端安装:

```bash
$ sudo npm install jn-wxapp -g
```

如何使用
--------------------------------------

在目的文件夹下执行命令：

```bash
$ wxxcx test
```

其中test是你的小程序项目名称
